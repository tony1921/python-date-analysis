## 📊 项目概述：帕默群岛企鹅数据可视化分析

本项目基于南极洲 帕默群岛（Palmer Archipelago）企鹅数据集（penguins.csv），通过数据清洗与多种可视化方法，分析不同企鹅物种、性别与岛屿等因素对其身体特征（体重、嘴峰尺寸、鳍长等）的影响关系。


#### 🎯 分析目标

清洗并整理企鹅数据

可视化不同物种之间的身体特征差异

探索性别对各种身体属性的影响

通过饼图、柱形图、PairPlot 等方式探索变量之间的关系


#### 📁 数据说明

原始数据包含 334 条企鹅样本，来自帕默群岛 3 个岛屿，主要字段包括：

species：企鹅物种（Adelie / Chinstrap / Gentoo）

island：所属岛屿

sex：性别

bill_length_mm / bill_depth_mm：嘴峰长度/深度

flipper_length_mm：鳍长

body_mass_g：体重

year：记录年份

项目对这些字段进行清洗（处理缺失值等）后用于分析。

Notebook 中对原始数据进行了：

缺失值检测与处理

生成清洗后的 cleaned_date 数据集

使用 pandas、numpy、seaborn、matplotlib 进行分析

## 🧾 总结与洞察

不同企鹅物种在体型与嘴峰尺寸上存在明显差异，可用于物种分类

性别同样是区分体重、鳍长等特征的重要因素

多变量关系呈现线性趋势，可支持进一步建模与预测
